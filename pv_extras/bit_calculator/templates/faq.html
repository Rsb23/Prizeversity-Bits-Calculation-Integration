What Is Bit Calculator?

Bit Calculator is a microservice created with Django and SQLite and hosted on PythonAnywhere. It allows university teaching assistants in Prizeversity classrooms the ability to calculate the amount of reward 'bits' without tedious calculations. 'Bits' are the currency of the Prizeversity ecosystem.

What Is CSV Normalization?

CSV Normalization is a process in which a CSV file is inputted and serialized. Specified data can then be normalized baed on pre-determined weights. The normalized data / CSV is then outputted.

How Do I Use Bit Calculator?

Start by making sure you are on the Bit Calculator tab by clicking the button labelled 'Bit Calculator' on the header from any page of the website. Input one or more CSV files from OnlineGDB that you got from downloading submission data (must be marked as a teacher on OnlineGDB). Find your lab section CRN or add a new CRN if it doesn't exist. Select criteria and click the generate button.
